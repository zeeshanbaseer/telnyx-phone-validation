# Telnyx Phone Validation Plugin for WordPress

A powerful WordPress plugin that validates phone numbers in your forms using Telnyx API. Compatible with WPForms, Contact Form 7, Gravity Forms, and other WordPress forms.

## Features
- Real-time phone number verification via Telnyx API
- Supports multiple form plugins: WPForms, Contact Form 7, Gravity Forms
- Customizable error messages for invalid numbers
- Lightweight and easy to configure
- Secure handling of Telnyx API key

## Installation
1. Upload `telnyx-phone-validation` to `/wp-content/plugins/`
2. Activate the plugin through WordPress dashboard → Plugins
3. Configure API key, form type, form ID, and phone field ID

## Benefits
- Reduce fake or invalid submissions
- Improve data quality
- Ensure accurate user contact information
- Seamless integration with WordPress forms
